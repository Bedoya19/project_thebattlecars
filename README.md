# project_thebattlecars
The Battle Cars: Proyecto personal de un juego de cartas en la web

## Que es "The Battle Cars"

Un proyecto sencillo de un juego de cartas sobre una pelea de carros usando armas y poderes absurdos.

Las cartas estan(estaran) divididas en 6 categorias. En donde los carros de las categorias mas altas son mas fuertes pero mucho mucho mas exagerado las armas.

Explicaciones detalladas vendran despues mientras se trabaja el el proyecto.
